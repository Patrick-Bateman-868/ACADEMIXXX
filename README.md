Код написан на языке программирования Python
Установите зависимости,прописав эту команду в терминеале:pip install Flask, Flask-SQLAlchemy, Flask-Login, Werkzeug
Архитектура и планы по улучшению обозначены .md
Чтобы запустить код,зайдите в app.py и нажмите запуск.Далее появится ссылка на сайт(локальный хостинг),в котором лежит проект.
